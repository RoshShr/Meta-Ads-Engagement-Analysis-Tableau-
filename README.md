# Meta Ad Performance Data Analysis - Tableau Project

This repository showcases my **Tableau Data Visualization** skills through the analysis of Meta Ads data. The project presents key insights on ad performance, user engagement, and campaign effectiveness. The visualizations highlight various aspects of Meta Ads, allowing stakeholders to make informed decisions regarding ad spend, targeting strategies, and performance optimization.

## Project Overview

The **Meta Ad Performance Data Analysis** dashboard leverages multiple datasets to provide a comprehensive view of Meta Ads performance across platforms like **Facebook** and **Instagram**. By analyzing metrics such as impressions, clicks, conversions, and demographic data, this project demonstrates my ability to work with large datasets, derive actionable insights, and present findings using Tableau.

### Key Metrics & Visualizations:
- **Total Ad Metrics**:
  - **Impressions, Clicks, Engagement Rate, Conversion Rate, and Purchases**: Provides a high-level overview of ad performance and ROI.
  - **Total Budget vs. Performance**: Analyzes the budget allocated to campaigns versus the actual results.
  
- **Engagement Analysis**:
  - **Engagement by Gender and Age**: Visualizations segment ad performance by demographics to help understand how different user groups engage with ads.
  - **Hourly Trends**: Shows how engagement varies by time of day, aiding in campaign scheduling optimization.

- **Geographic & Time-based Insights**:
  - **Geo HeatMap**: Displays geographic distribution of user interactions with ads, highlighting high-performing regions.
  - **Heatmap**: Tracks user engagement over weeks, providing insights into seasonal or time-based trends.
  
- **Event-based Performance**:
  - **Event Type Segmentation**: Allows filtering and analysis by event types like Likes, Comments, Shares, and Purchases.
