
# Meta Ads Data Analytics

This repository contains raw datasets for Meta Ads data analysis.

## Datasets

1. **ads.csv**: Contains ad-specific details such as ad ID, campaign ID, impressions, clicks, and conversions.
2. **campaigns.csv**: Contains data related to ad campaigns, including campaign ID, name, budget, start date, and end date.
3. **users.csv**: Contains user data including user ID, age, gender, location, and device type.
4. **ad_events.csv**: Contains event-level data for ads, including event ID, ad ID, user ID, event type (click, view, etc.), and timestamp.

These datasets are intended for analysis and visualization, and can be used with tools like Tableau to explore ad performance and user engagement metrics.
